Tag Meaning
background-color Used to set the color of the background of a element
height Is responsable for setting the height of a element
width Is responsible for setting the width of a element
padding Padding sets the amount of space that's between a object and its orbit (area around it).
font For setting the attributes of font like if it's bold, italic or underlined ect.
font-family Sets what the style of text will be used such as "Times new Roman" or "Ariel".
font-size: Is what you use to set how big or small you want your text to be.
text-align: Places your text in different prefered locations such as left aligned, centered or right aligned.
border-style Is a tag that resets the previous styles of a class. It's useful to make web developers edit new classes without having to look through tonnes of previous code that will affect the new class.
all:unset Is a tag that resets the previous styles of a class. It's useful to make web developers edit new classes without having to look through tonnes of previous code that will affect the new class.
display:flex Is the building block for using flex commands. It's used to tell the compiler you're about to use flex comannds.
flex-direction Sets the layout of data. For example you can say flex-direction:row to show information side by side, and you can use flex-direction:column to show information underneath each other.
HTML
Tag Meaning
head "head" is the tag used to contain metadata. This is for information about the html document such as it's styles, links and scripts.
header header "header" is the tag used for placing information in the upper section of a website known as the header.
body The Body tag is used for the middle section of your website which usually holds the bulk of your website's features.
footer "footer" is the tag used to represent the bottom section of your website. It will usually have limited information which may include contact details
p "p" is known as the paragraph tag which is used for storing paragraphs of information along with images if a image tag is placed inside of it.
img The "img" tag is used to embed images into your website via a hyperlink or a picture saved to your computer
h1, h2, h3, h4 h1,h2,h3,h4 These tags are special pre-set styles usually used for headings.You can apply them to various text by typing between the open and closing of the tags
ul "ul" stands for unordered list. This tag is ment for setting up lists that follow no order like bullet points.
ol "ol" stands for ordered list. This tag is for data that follow a logical order . example 1,2,3,4.
li li "li" stands for list item. This tag is used to set what's inside of your unordered lists or ordered lists.
